# Quick Pins — расширение для Chrome

Минимальный менеджер ссылок во всплывающем окне. Сохраняет URL с подписью, отображает favicon, хранит данные в `chrome.storage.sync`, поэтому ссылки синхронизируются между браузерами под одним аккаунтом.

## Структура

-   [`manifest.json`](manifest.json): описание расширения.
-   [`popup.html`](popup.html): разметка всплывающего окна.
-   [`styles.css`](styles.css): минималистичный UI.
-   [`popup.js`](popup.js): логика добавления/удаления ссылок.

## Установка в Chrome (режим разработчика)

1. Скачайте папку проекта целиком (или склонируйте репозиторий).
2. Откройте `chrome://extensions/` в Chrome.
3. Включите **Режим разработчика** (правый верхний угол).
4. Нажмите **Загрузить распакованное** → выберите папку проекта.
5. Иконка расширения появится в панели. Закрепите её через меню Extensions.

## Использование

1. Откройте всплывающее окно (иконка «Quick Pins»).
2. В поле URL вставьте адрес сайта. Необязательно добавьте подпись.
3. Нажмите «Добавить». Ссылка появится ниже, выводится favicon сервиса.
4. Клик по записи открывает сайт в новой вкладке, крестик удаляет ссылку.

## Как это работает

-   При инициализации [`popup.js`](popup.js#L1) загружает `items` из `chrome.storage.sync` и рендерит список.
-   Форма отправляет новые данные, нормализует URL, перезаписывает `items`, хранит их синхронно и обновляет DOM.
-   Для favicon используется публичный сервис Google (`/s2/favicons`) по домену.

## Кастомизация

-   Измените палитру или отступы в [`styles.css`](styles.css#L1).
-   Можно добавить drag-and-drop reorder или авто-подстановку текущей вкладки, подключив `tabs` и `activeTab` permissions.
